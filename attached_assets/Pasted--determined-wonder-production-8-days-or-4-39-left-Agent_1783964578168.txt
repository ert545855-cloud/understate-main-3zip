yol — Nixpacks'i zorla Node kullanmaya it. Repo köküne nixpacks.toml adında küçük bir dosya ekle, içeriği şu olsun: tomlproviders = ["node"] [phases.install] cmds = ["npm install"] [start] cmd = "node server/main.js" Bu, Nixpacks'e "burada Python falan yok, sadece Node kullan" diyor.
